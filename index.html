What is TARDIG?

Typography and Responsive Design in Grids

TARDIG is a front-end boilerplate designed to be a starting point in the creation of web pages.

See Full Documentation

Tardig comes in a vanilla CSS and an indented Sass format. Soon, it will be updated to support SCSS, Less, and Stylus as well.

Normalize.css

Normalize.css is implemented to create browser consistency.

Grid System

The grid system is fluid by default but also comes with a responsive flavor.

Typography

Typography uses a modular scale and a formula to determine smart line heights.

Dev Assistance

Dev Assistance is a set of tools to help troubleshoot errors in coding.
