Using Sass Mixins

Like variables, each component folder has a file called #[code _mixins.sass} that creates the functionality of the boilerplate. Each one can be edited to fit your needs, but the ones that will be the most useful to your custom sheets are the breakpoint mixins.

Mixin Mininum width
break-xs 480px
break-sm 768px
break-md 992px
break-lg 1200px

To use these mixins in indented Sass you can write #[code +break-**}, and then write your custom styles indented below it, rather than typing out #[code @media only screen and (min-width: *)}.
